#### 4.1 工具方法 `Util`

  ```javascript
    1. $.util.deparam(urlParam);                            // 地址参数反序列化

    4. $.util.post(url, [param], [success], [unsuccess]);   // 统一处理 post 请求，使之符合业务需求
    5. $.util.render(tmpl, json, [options]);                // 渲染模板
  ```
